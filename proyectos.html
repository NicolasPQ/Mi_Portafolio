Sistema de Gestión y Ventas para el Restaurante "TuEx"

En aras de modernización el restaurante “TuEx” solicitó un sistema que le permitiera realizar ventas, mantener un registro de éstas, gestionar a sus trabajadores, clientes, insumos, mesas, proveedores, menús y que pudiera realizar reportes de las órdenes. Posteriormente a un análisis del restaurante, se estableció un modelo de negocios, en base a éste se diseñó una base de datos relacional en SQL Server para desarrollar un software en Visual Studio 2022 que permitiera satisfacer todas las necesidades del restaurante. El software desarrollado es de escritorio y cuenta con control de acceso y restricción de funciones según usuario; permite realizar órdenes, gestionarlas e imprimirlas; y puede realizar la administración de todas las necesidades anteriormente mencionadas.
